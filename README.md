# **thehumanserver.org**
Hiya! Karis here. This repository is dedicated to THE HUMAN SERVER and various subsidiary projects.

## Direction
My main project right now is The Human Server. I want it to be an archive for written works from <a href="https://en.wikipedia.org/wiki/Yoko_Taro">Yoko Taro.<a>\
I will slowly add resources to learn more about his works over time. Feel free to submit anything you would like added ! You will receive credit.\
I also have *THS-Archive*, a tag-based image hosting service. WARNING: I haven't updated it in months, and it's currently under lockdown. If you're interested in doing that for me, please reach out !!
